## 3장

- AWS IAM : 권한 관리 서비스
- 아마존 EC2: AWS에서 제공하는 가상 클라우드 서버
- 아마존 RDS: 다양한 데이터베이스 엔진을 제공하는 관계형 데이터베이스
- 아마존 S3: 객체 스토리지 서비스

- AWS IAM 살펴보기
- IAM 사용자, IAM 그룹, IAM 정책, IAM 역할로 이루어져 있다.

## 4장

- EC2란 Amazon Elastic Compute Cloud의 약자로 AWS에서 제공하는 가상 클라우드 서버를 의미한다.
- EC2는 AMI, 인스턴스 유형, 스토리지 옵션, 보안 그룹, 키 페어 등으로 이루어져 있다.
- AMI는 Amazon Machine Image의 약자로 아마존 EC2를 시작하는 데 필요한 소프트웨어 구성을 담고 있는 템플릿이다.
